python 广播

广播的原则：如果两个数组的后缘维度（trailing dimension从末尾开始算起的维度）的轴 长度相符或者其中一方为1，那么认为是广播兼容的。广播会在缺失和（或）长度为1的维度上进行。

根据广播的原.则，较小数组的广播维必须为1。于是就有了一个很普遍的问题，即为了广播专门增加一个新的长度为1的维度。
